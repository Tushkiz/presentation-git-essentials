how do i submit the changes?

adding remote

$ git remote add origin git@github.com:[username]/[repo-name].git

submitting changes

$ git push origin master

cloning

$ git clone git@github.com:[username]/[repo-name].git

getting changes from remote

$ git pull

getting changes from remote like a boss

$ git pull --rebase

history

$ git log
$ git log --stat

basic workflow

- make changes

- stage the changes (git add)

- review the changes (git status, git diff)

- commit the changes (git commit)

- get/submit the changes (git pull, git push)

advanced git

branching out

$ git branch clean-up

switching branches

$ git checkout clean-up

switching back to master

$ git checkout master

merging changes

$ git merge clean-up

deleting branch

$ git branch -d clean_up
